Зачем вам создавать игру на js?

Почти каждый задумывался о том, что бы начать карьеру програмиста. И почти перед каждым ставал вопрос "А на каком языке я буду писать код?" Многие выберут C, C++, но есть и другие.

Тот, кто интересовался работой сайтов, знают, что есть несколько языков программирования, которые могут стать вашими инструментами во время создания сайта. Один из самых популярных сейчас - JavaSript.
JavaSript, или сокращенно js, выступает именно тем инструментом, который может добавить функционала вашему сайту.

В наше время, очень много сайтов, которые помимо HTML и CSS используют JS для того, что бы добавить определенных функций своему сайту.

Так с чего же начать?
Начинайте с самого начала. Изначально вам будет интересно написать самую первую программу, которая приоткроет вам завесу мира, который может вам открыть JS.
Эта программа являэтся первой, которую можно написать на любом языке программирования и называется она "Hello, world" и состоит всего лишь из одной строчки кода: 

                            
console.log("Hello, world!");

В дальнейшем, когда вы уже освоите основы JavaSript, вам захочется усовершенствовать ваши навыки и вы подумаете, а что же можно написать такого, что поможет мне развить себя?т
Вот тут то вы и подумаете о написании простенькой игры, которая совмещала бы в себе основы js, немного HTML и чуточку CSS.

После того, как у нас есть мяч и поле мы можем приступать к их анимации.

Комфортное отображение

Итак, первым делом составим список, что же нам надо сделать для комфортной игры?

  • Задать функцию события
  • Задать координаты относительно окна браузера
  • Задаем координаты мяча
  • задаем координаты поля
  • Запрещаем мячу пересекать приделы поля

После этого приступаем к первому пункту нашего плана и задаем функцию события,
которая будет выглядеть примерно так:

 
                            
field.onclick = function(event) {}

Итак, теперь мы можем приступить к выполнению следующего пункта плана.

Задаем координаты

Теперь нам нужно сделать так, что бы поле комфортно отображалось в браузере.
Для этого внутри нашей функции события прописываем вот такую строчку кода: 

                            
let fieldCoords = this.getBoundingClientRect();

Теперь предлагаю задать координаты нашему полю и мячу. Все это будет выглядеть примерно так:

                            
let ballCoords = {
    top: event.clientY - fieldCoords.
    top - field.clientTop - ball.clientHeight / 2,
    left: event.clientX - fieldCoords.
    left - field.clientLeft - ball.clientWidth / 2
};

Отлично! Этот шаг выполнен, а значит, мы можем переходить к следующему!

А где же рамки?

Вот мы и приступаем к последним шагам создания нашей игры.
Для того, что бы наш мяч не смог вылететь за пределы игрового поля, предлагаю прописать вот такую комбинацию кода: 

                            
if (ballCoords.top < 0) ballCoords.top = 0;

if (ballCoords.left < 0) ballCoords.left = 0;

if (ballCoords.left + ball.clientWidth > field.clientWidth) {
    ballCoords.left = field.clientWidth - ball.clientWidth;
}

if (ballCoords.top + ball.clientHeight > field.clientHeight) {
    ballCoords.top = field.clientHeight - ball.clientHeight;
}

И теперь добавляем последний штрих:

                            
ball.style.left = ballCoords.left + 'px';
ball.style.top = ballCoords.top + 'px';

Вот и все! Если вам интересно, как будет выглядеть весь код, то вот:
